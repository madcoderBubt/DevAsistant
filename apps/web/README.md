# Dev Assistant - Web Application

This is the frontend React application for the Dev Assistant project.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Code Editor**: Monaco Editor (@monaco-editor/react)
- **Icons**: Lucide React

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run preview`: Previews the production build locally.

## Features Included

- Code Formatter
- Diff Tool
- JSON Parser
- Unit Conversion
- Image Converter
- Data Converter
